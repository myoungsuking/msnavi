# 원본 데이터 (행정안전부 자전거길 자료)

본 폴더는 **행정안전부가 공개한 국토종주 자전거길 공식 자료**입니다.
공공데이터이므로 그대로 커밋합니다. ETL 스크립트의 입력으로 사용됩니다.

| 파일 | 원본명 | 설명 | 인코딩 |
| --- | --- | --- | --- |
| `gukto_routes.csv` | ★국토종주 자전거길 노선좌표.csv | 노선별 위경도 좌표 시퀀스 (53,418행) | CP949 |
| `gukto_pois.csv` | ★국토종주자전거길 주변시설 좌표정보.csv | 인증센터/화장실/급수대/공기주입기 (1,133행) | CP949 |
| `route_codebook.xlsx` | ★노선정보 코드북.xlsx | `ROAD_SN`(1~46) ↔ 노선명 매핑 | XLSX |

## 스키마

### gukto_routes.csv
| 컬럼 | 설명 |
| --- | --- |
| `순서` | 노선 내 점의 순번 |
| `국토종주 자전거길` | `ROAD_SN` (노선코드, 1~46) |
| `위도(LINE_XP)` | 위도 |
| `경도(LINE_YP)` | 경도 |

### gukto_pois.csv
| 컬럼 | 설명 |
| --- | --- |
| `구분` | 인증센터 / 화장실 / 급수대 / 공기주입기 |
| `이름` | 인증센터는 개별명, 그 외는 소속 노선명(예: "아라길") |
| `경도`, `위도` | 좌표 |

### route_codebook.xlsx
| 컬럼 | 설명 |
| --- | --- |
| `ROAD_SN` | 노선 코드 |
| `국토종주 자전거길 정보` | 노선명. "현재 노선변경으로 사용하지 않음" 은 제외 대상 |

## 적재 방법

```bash
# Postgres(PostGIS) 가 기동되어 있고 .env 설정이 끝난 상태에서
cd apps/api
npm run import:official
```

자세한 로직은 `apps/api/src/scripts/import-official.ts`.
